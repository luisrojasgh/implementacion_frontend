Colores

El tipo de datos CSS denota un color en el sRGB color space. Un color puede ser descrito de cualquiera de las siguiente maneras: Usando una palabra clave. Usando el sistema RGB cubic-coordinate (usando #-hexadecimal o las notaciones funcionales rgb() y rgba() ) Usando el sistema HSL cylindrical-coordinate (usando las notaciones funcionales hsl() y hsla()) Hay que destacar que la lista de valores de color aceptados ha ido creciendo conforme la especificación evolucionaba, para acabar con la lista de colores de CSS3. Asociado con el color en el espacio sRGB, un valor también consiste en una coordenada alpha-channel o valor de transparencia, indicando cómo se debe composite o mezclar el color con su color de fondo. Aunque los valores de colores en CSS son definidos de manera precisa existe la posibilidad de que parezcan distintos en dispositivos diferentes. La mayoría de ellos no están calibrados y algunos navegadores no soportan los color profile de algunos dispositivos de salidas. En esta situación el color puede variar bastante.
